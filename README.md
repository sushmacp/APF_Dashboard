# APF Dashboard

APF Dashboard Boilerplate 

## Installation

Clone the repository
```bash
 https://github.com/chethan-aritha/apf_dashboard.git
```

Navigate to project folder
```bash
cd apf-dashboard
```

Install apf-dashboard with npm
```bash
 npm install
 npm start
```
